# ShiftGuard10-Robust-Image-Classification
Robust deep learning framework for 10-class image classification under distribution shift conditions using custom CNNs with Residual Connections and SE Attention blocks. Applied CutMix, MixUp, SWA, Ensemble Learning, and TTA in PyTorch, improving Macro-F1 score from 0.62 to 0.91+.
## Dataset
Kaggle ShiftGuard10 Robust Image Classification Challenge
https://www.kaggle.com/competitions/shift-guard-10-robust-image-classification-challenge
